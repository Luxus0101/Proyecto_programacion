-----------------------------
PROGRAMMING STYLE GUIDELINES
-----------------------------

-Variables and functions MUST be defined with lowercase letters and using "_" when separation is needed:
  int mi_variable = 0;
  char * mi_funcion();

-Macros and constants MUST be defined using uppercase letters:
  #define CHAR_LENGTH 40
  const int VARIABLE_CONSTANTE;

-Until github starts being used, all changes should be documented in a separated log file using the following format:
  Author: Lucas Piorno <lucas.piorno@estudiante.uam.es>                   /*Name and email of author*/
  Date:   Mon Feb 15 01:14:39 2021                                        /*Date of the changes*/
  Description: corrected the formatting of file game.c                    /*Brief description of changes*/

-Comment each header file according to the presentation in Moodle
